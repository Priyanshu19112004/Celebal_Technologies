# 🚀 Data Exploration & Cleaning Using Pandas

## 📌 Project Overview
This project demonstrates the use of **Python Pandas** for performing real-world data exploration and cleaning operations on a CSV dataset. The dataset was analyzed, cleaned, and transformed by handling missing values, removing duplicate records, filtering important data, and creating derived columns for better insights.

This work was completed as part of the **Celebal Technologies Internship Program** and showcases fundamental data preprocessing techniques used in **Data Analytics** and **Data Engineering** workflows.

---

# 🎯 Objectives
✔ Load CSV dataset into Pandas DataFrame  
✔ Explore dataset structure and information  
✔ Detect and handle missing values  
✔ Remove duplicate records  
✔ Perform filtering and column selection  
✔ Create derived columns for analysis  
✔ Export cleaned dataset into CSV format
