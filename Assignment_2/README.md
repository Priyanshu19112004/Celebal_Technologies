# 📊 SQL Assignment 2: Data Analysis Using Filtering, Aggregation and Business Queries

## 🚀 Assignment Overview

This assignment demonstrates the practical application of SQL for business data analysis using the Superstore Sales Dataset.

The objective of this assignment is to analyze transactional sales data by applying filtering, aggregation, sorting, trend analysis, and data validation techniques to generate meaningful business insights.

---

## 🎯 Assignment Objectives

- Explore and understand the dataset structure.
- Apply SQL filtering techniques using WHERE clause.
- Perform aggregation using GROUP BY functions.
- Analyze sales performance across regions and categories.
- Identify top customers and products.
- Evaluate monthly sales trends.
- Validate data quality and consistency.
- Generate business-oriented insights.
